@magic-modules/module-list

this is the @magic-modules ModuleList component. It shows a list of all @magic-modules.

installation

npm install @magic-modules/module-list

usage

in a page or module View

export const View = () => ModuleList()

@magic-modules

modules are the grimoires of @magic. modules add both client and server functionality to your @magic app.

  • @magic-modules/accordion

    Accordion shows a list of items that horizontally open/close.

    docs / demo

  • @magic-modules/hero

    Hero modules are full screen top of page content.

    docs / demo

  • @magic-modules/language-switch

    LanguageSwitch provides file hierarchy based multilanguage support for magic.

    docs / demo

  • @magic-modules/gdpr

    Gdpr compliant popup that allows users to allow or deny cookies.

    docs / demo

  • @magic-modules/pre

    Pre allows you to display javascript code with syntax highlighting.

    docs / demo

  • @magic-modules/sound-cloud

    embed soundcloud track, playlist and user widgets

    docs / demo

  • @magic-modules/video-embed

    VideoEmbed embeds videos from any video hoster using iframes. comes with default support for vimeo and youtube.

    docs / demo

  • @magic-modules/git-badges

    GitBadges shows a list of github repository status badges.

    docs / demo

  • @magic-modules/git-list

    show a list of git repositories like the one you are looking at.

    docs / demo

  • @magic-modules/module-list

    ModuleList shows the list of @magic-modules you are looking at.

    docs / demo

  • @magic-modules/theme-list

    ThemeList shows a list of all @magic-themes.

    docs / demo

  • @magic-modules/library-list

    LibraryList shows a list of all @magic-libraries libraries.

    docs / demo
